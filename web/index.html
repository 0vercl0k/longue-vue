Longue vue

The «Longue vue» exploit targets NETGEAR DGND3700v2 modem / routers. From the Internet or the LAN, an attacker can execute arbitrary commands as root on your router; fully controlling your device.

⚠ Clicking the button tests if you are vulnerable by running the exploit: it will dump your credentials as well as the Linux kernel version running on the device.
